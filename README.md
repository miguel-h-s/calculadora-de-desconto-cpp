# Calculadora de Desconto em C++

## Sobre o projeto
Este é um programa simples em C++ que:

- Solicita o nome do produto, preço e desconto  
- Valida os valores inseridos (não permite negativos ou desconto maior que o preço)  
- Calcula e exibe o preço final do produto  

Ideal para quem está começando a aprender programação em C++.


## O que aprendi
- Entrada e saída de dados (`cin` e `cout`)  
- Condicionais (`if` / `else`)  
- Estrutura básica de um programa C++ (`int main()`)  
- Uso de variáveis  
- Retornos de função (`return 0;` e `return 1;`)
